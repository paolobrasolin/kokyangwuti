Conway's Game Of Life

This is an implementation of the Game Of Life. GOL is a cellular automaton devised by John Horton Conway, an amazing mathematician and unfortunately yet another victim of fricking COVID-19.

This toy allows you to customize every crucial aspect of the simulation by writing your own JS code. The buttons in each section will load interesting defaults.

When you're done editing the code you must restart the simulation.

Automaton rule

The heart of the automaton is the rule which calculates the evolution of the system. Conway's traditional rule is the most famous one. An interesting alternative is a neighbours parity rule which is able to produce pretty fractal patterns (try using it with a seed of a single point).

Initial seed

The initial conditions of the field completely determine its evolution. An obvious choice is filling it with random cells. Another interesting choice (when used with the parity rule) is starting with a single lone cell.

World Boundaries

The simulated world is finite and we must decide how to handle calculations at the boundaries. We can wrap around the edges but also treat them like an impenetrable wall.
